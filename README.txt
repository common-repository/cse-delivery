Модуль интеграции Вашего интернет-магазина WooCommerce со службой доставки Курьер Сервис Экспресс: расчет стоимости доставки по России (до дверей и в пункт самовывоза), формирование накладных и заказов на доставку, отслеживание грузов и получение статистики взаимодействия Вашего магазина с перевозчиком.

== Установка ==

1. Скачайте модуль с официального репозитория WordPress и установите стандартным способом
2. Заключите договор обслуживания с Курьер Сервис Экспресс
3. Настройте модуль, в соответствии с условиями договора

== Frequently Asked Questions ==

= Как работает модуль? =

Для полноценной работы модуля Вам необходимо заключить договор с Курьер Сервис Экспресс и настроить его, в соответствии с условиями договора.

= Что делать, если модуль не считает стоимость доставки? =

Убедитесь, что настройки модуля соответствуют договору. При возникновении сложностей обратитесь к Вашему менеджеру КСЭ

== Скриншоты ==
